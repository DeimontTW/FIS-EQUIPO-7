# Priorización de los requisitos funcionales #
<font color = green>**Prioridad alta**</font>   
<font color=yellow>**Prioridad media**</font>    
<font color=red>**Prioridad baja**</font>

* <font color = green>*El mapa debe representar la infraestructura de la facultad de matemáticas en un esquema detallado en 2D y un entorno virtual Street view*</font>: Este requisito es el más indispensable pues forma el eje central del sistema

* <font color = green>*Mostrar la ubicación del usuario y rastrearla usando el GPS de su dispositivo*</font>: Importante para que el usuario tenga un punto de referencia y así poder guiarse, mejorando su experiencia por la facultad

* <font color = green>*El entorno Street View deberá contar con botones de navegación en forma de flechas para permitir la navegación del usuario (arriba, abajo, izquierda, derecha)*</font>: Esencial para que el usuario pueda recorrer el entorno y conocerlo

* <font color = green>*Mostrará por medio de leyendas y símbolos puntos de interés de la facultad, representados por colores*</font>: Importante para que el usuario pueda conocer los lugares de la facultad e identificarlos

* <font color = yellow>*Alternar entre la vista en 2D y el entonrno Street View en una transición cuando el usuario así lo desee*</font>: Necesaria para que el usuario se familiarize con el mapa y ofrecer una buena experiencia de uso

* <font color = yellow>*Incluirá una heramienta de búsqueda para puntos de interés y planificar una ruta con punto de inicio y destino*</font>: Esencial para evitar la desorientación y facilitar la búsqueda y la ruta a seguir que el usuario necesite

* <font color = yellow>*El mapa enviará notificaciones push cuando ya se encuentre abierta la cafetería, sala de cómputo, biblioteca, entre otros puntos de interés*</font>: Requerimiento para mantener informado al usuario sobre eventos que ocurran en determinado momento notificándolos de manera atractiva

* <font color = red>*Apartado de configuración para activar o desactivar tooltips y animaciones del mapa*</font>: Para simplificar el uso del mapa si el usuario así lo desea

* <font color = red>*Opción de traducción de idioma (Español-Inglés)*</font>: Permite a usuarios que hablen el idioma inglés usar el mapa
